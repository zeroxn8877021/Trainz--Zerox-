About Us

What is ZEROX STUDIO?

ZEROX STUDIO is India's premier community dedicated to creating and sharing Trainz Simulator assets. We build high-quality 3D models of Indian Railways rolling stock — from vintage steam locomotives to modern EMUs — completely free for the community.

Every addon is created with love for the Indian Railways heritage and built to the highest standards of accuracy and game compatibility.

Authentic Indian Railways

Every model faithfully recreates real Indian Railways rolling stock with accurate liveries, dimensions and details.

100% Free Downloads

All addons are completely free. No paywalls, no subscriptions — just pure passion for Trainz.

Community Driven

Built by enthusiasts, for enthusiasts. Join thousands of Indian Railways fans in the Trainz community.

What is Trainz Simulator?

Trainz: A New Era / Trainz Railroad Simulator is a train simulation game available on PC and Android. It lets you drive trains, build routes, and install custom assets created by the community.

On Android, Trainz Simulator provides a rich mobile experience where you can drive iconic locomotives across detailed routes.

Guide

How to Install Addons

01

Download the Addon

Click the Download button on any addon page. The file will be a .cdp or .tzarc package.

02

Open Trainz Content Manager

Open Trainz on your device and navigate to the Content Manager or My Content section.

03

Install the Package

Select "Install from File" and browse to the downloaded addon package to install it.

04

Enjoy!

Your new addon is ready. Find it in the Trainz asset library and start driving!
